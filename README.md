Customer Segmentation Project

A complete end-to-end customer segmentation project using Python and Machine Learning.

## Project Overview
Analysed 500+ customers and grouped them into distinct segments using K-Means clustering,
based on behavioural and demographic data. Each segment received a tailored business strategy.

## Project Phases
- Phase 1 — Exploratory Data Analysis (EDA)
- Phase 2 — Data Preprocessing & RFM Analysis
- Phase 3 — K-Means Clustering (Elbow Method + Silhouette Score)
- Phase 4 — 10 Visualisation Dashboards
- Phase 5 — Business Recommendations + PDF Report

## Tools & Technologies
- Python 3
- pandas, numpy
- scikit-learn (KMeans, PCA, StandardScaler)
- matplotlib, seaborn
- fpdf2 (PDF generation)
- Google Colab

## Key Results
- Identified 4 distinct customer segments
- Built RFM scoring model for each customer
- Generated 10 visualisation charts including scatter plots, heatmaps, radar charts, PCA plots
- Produced a full business recommendation report with marketing strategies and KPIs per segment
